# البداية السريعة مع Stampcoin Platform

هذا الدليل سيساعدك على إعداد وتشغيل Stampcoin Platform على جهازك المحلي.

## المتطلبات

- Node.js 16.x أو أحدث
- npm أو yarn

## الإعداد

### 1. تثبيت المشروع

```bash
# استنسخ المستودع
git clone https://github.com/your-username/stampcoin-platform.git
cd stampcoin-platform

# تثبيت التبعيات
npm install
```

### 2. تشغيل الخادم

```bash
# بدء الخادم في وضع الإنتاج
npm start

# أو بدء الخادم في وضع التطوير مع إعادة تحميل تلقائي
npm run dev
```

سيتم تشغيل الخادم على المنفذ 3000 بشكل افتراضي. يمكنك الوصول إلى الخادم عبر: http://localhost:3000

## أول طلب

بعد تشغيل الخادم، يمكنك إجراء أول طلب للحصول على حالة الخادم:

```bash
curl http://localhost:3000/api/status
```

الرد المتوقع:

```json
{
  "status": "running",
  "timestamp": "2026-02-17T12:00:00.000Z",
  "version": "2.0.0"
}
```

## المتغيرات البيئية

يمكنك تعيين المتغيرات البيئية عن طريق إنشاء ملف `.env` في جذر المشروع:

```env
PORT=3000
NODE_ENV=development
API_SECRET=your-secret-key
```

## التكوين الافتراضي

يمكنك تعديل إعدادات التطبيق في ملف `config/default.json`.
